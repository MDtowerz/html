Red Handed

(NOTE:   Webgl version may have problems. Please try the windows version if possible )

Press R to reload a level if any problem occurs

Press Y to invert Y-axis at any time

Controls are explained in-game

The web version is ver 1.1 which is also available for download. Along with the original jam version


" A stealth game where you have to get caught "

Originally made in under 3 days for the Wowie game jam where the theme was " Failure is progress " and then later added to 7dfps

Sound effect for the gun is a modified version of a sound from https://freesound.org/


-> Along with WASD, the game supports ZQSD for AZERTY keyboard users
